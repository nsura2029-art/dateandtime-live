02

Seven-segment

How the display works.

Each digit is a div with seven absolutely-positioned children. Each segment is a span with a hexagonal clip-path. To light up a number, we toggle the on class on the right segment names. The 7-segment pattern is hard-coded — no font, no SVG.

  • Digit patterns
    0: [a,b,c,d,e,f] · 1: [b,c] · 2: [a,b,g,e,d] · 3: [a,b,c,d,g] · 4: [f,g,b,c] · 5: [a,f,g,c,d] · 6: [a,f,g,e,c,d] · 7: [a,b,c] · 8: [a,b,c,d,e,f,g] · 9: [a,b,c,d,f,g]
  • Segment shapes
    Horizontal: clip-path: polygon(10% 0, 90% 0, 100% 50%, 90% 100%, 10% 100%, 0 50%) — gives the pointy-end hexagon look. Vertical: similar but rotated.
  • The "off" state
    In dark mode inactive segments drop to opacity: .06 — that's what makes it look like a real LED clock instead of a "always-on" font.
  • Glow
    Light: drop-shadow(0 5px 8px rgba(91,77,145,.10)) for a soft purple shadow. Dark: drop-shadow(0 0 12px rgba(249,115,22,.25)) for the LED amber glow.
  • Animation
    requestAnimationFrame at 60fps. The hundredths digit changes 100 times per second, the seconds digit changes once a second.
  • Theming
    One data-theme="light" | "dark" attribute on the <html> element. CSS variables flip. localStorage persists. OS preference is respected on first visit.

    Reuse

    Where this clock is going.

    One <SevenSegClock> component, used in five pages. Each page forces its own theme. The component itself doesn't care about which page it's on.

    • USE 01

      Today · the hero clock

      The full-size 7-segment display as the page hero. Light mode for the public landing, dark mode for the logged-in dashboard. Updates every frame.

    • USE 02

      World Clock · inline city times

      A smaller variant for each pinned city. Click a city → it lights up briefly to confirm the selection. Hovering a row keeps the time frozen so you can read it.

    • USE 03

      Meeting Finder · the overlap time

      The result of a 3-city search is the suggested meeting time. Big clock, with the city's timezone underneath. Dark mode forced — keeps the focus on the time.

    • USE 04

      Timezone Navigator · the from/to times

      Two clocks side by side, one for "from" and one for "to". When the user drags the time slider, both update. Light mode here, since it's a working/utility surface.

    • USE 05

      3D Globe · the hover tooltip

      Hovering a city on the globe shows a small 7-seg clock with that city's current time. Tooltip-scale (smaller digits, no date line, no AM/PM pill).
