City Page Templates — 4 Designs for 2030

Light + dark mode · Live time · Tampa, FL demo

Four distinct design directions for /world-time/city/{slug}/. Each is a fully self-contained HTML file with inline CSS + JS. All share the same internal link graph, the same JSON-LD structure, and the same live-time update pattern. Template D is the recommended default — it follows timeanddate.com's data-density pattern but redesigned for 2030, follows our design system (no card aesthetic), includes 17 sections + 2 ad slots, and is SEO-optimized.

Recommended rollout

  1. Ship D (Data Hub) as the default — TAD-density without the card aesthetic, best for SEO + ads, follows design system + light/dark
  2. A (Bento) for A/B testing on top 1,000 cities — see if users prefer bento over linear
  3. B (Dashboard) for power-user landing (tool-heavy users returning often)
  4. C (Story) for top 100 cities only — long-form SEO moat where Wikipedia-style content matters most

Why D is the recommended default

  • Data density — users on timeanddate.com come to GET DATA. TAD layout is proven for that. 8 color-coded info blocks, 10 quick-info pills, 17 sections.
  • No card aesthetic — follows site design system (rows, lists, grids) instead of bento cards. Color-coded tints instead of card backgrounds.
  • SEO-optimized — 17 H2 sections means Google has 17 ranked targets per page. Long-form internal link grid in footer.
  • Ad-friendly — 2 explicit ad slots (728×90 leaderboards) between sections. Natural break points at end of section 5 (time diff) and end of section 10 (holidays).
  • Light + dark via CSS variables — same as other templates, with warm rust primary that feels less cold than TAD's blue.
  • Editorial people from C — reuses the "People Tampa gave the world" portrait grid that gives emotional weight.
  • Day-arc from A — keeps the sun visualization (one of the best visual hooks for time data).

How to preview

  • Click the sun/moon icon (top-right) to toggle dark mode
  • The clock updates in real time (60fps via requestAnimationFrame)
  • Click any city card / link to test navigation
  • Resize the window to see responsive breakpoints

All 4 templates are self-contained single-file HTML. No build step. No dependencies. Drop into the site when ready.
